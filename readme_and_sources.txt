Projekt "ZipLineVR"
==================

Diese Applikation ist zur Verwendung im TU Wien JumpCube konzipiert.


Program- und Framework Versionen:
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
Unity:			2018.2.20f1
SteamVR:		2.0.1
Gaia:			1.9.1-c1
SocketIO for Untiy:	1.0.1	


Zielsystem:
^^^^^^^^^^^
Intel Core i7-6700K @ 4.0 GHz
16GB DDR4 RAM
GeForce Titan XP (Driver 416.34)
HTC VIVE (non-pro)
Windows 10 Pro N (64 Bit), Build 1607
SteamVR Client v1515522829 (09.01.2018)
